# DB_Project
2023년 2학기 데이터베이스 프로그래밍
주제는 지식인 사이트를 만들고 ChatGPT를 사용한 자동 답변 구현이다. 
1. ChatGPT API를 이용해 자동 답변 생성
2. 내공 시스템을 이용한 등급제
3. 파일 입출력 구현


# 📝구현 과정에서의 특징

2. 우리는 프론트엔드가 없으므로 정적 컨텐츠 HTML + Thymeleaf로 개발하였다.
